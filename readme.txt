Sholi the Ultimate URL Shortener allows you to generate Short URLs with your wordpress website for posts, pages, attachments and custom posts.

== Description ==

**Sholi The Ultimate URL Shortener** Automatically generate short urls from your wordpress site when publishing new page, post.

Sholi URL Shortener allows you to connect your WordPress Website to the Sholi API via Bearer token.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `sholi-url-shortener.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Goto Tools then select Sholi URL Shortener
1. Copy the access token from sholi's dashboard and start generating your Short URLs


== Features ==

* Generate Short URLs within your wordpress site.
* Share Sholi Short link from your Post List and Post Edit page.
* Short URL Shortener has Support for Custom Post Type.
* Generate Sholi link of your old posts with just one click.
* Option to choose from your list of post types.
